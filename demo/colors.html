Usage

This color palette is mixture of semantic's base colors and material design. Each of these colors is defined with a base color class and optional lighten, darken and accent classes. This system provides a consistent and robust color palette without the hassle of working with hex values.

Background Color

To apply a background color, just add the color name prefixed with bg- to the element as a class. The optional shade classes must also be prefixed with bg-.

<div class="bg-teal bg-lighten-2"></div>
Selectable

You can make a background appear selectable by adding the class bg-selectable. This will change the color on both :hover and :active states.

<div class="bg-teal bg-lighten-2 bg-selectable"></div>

Text Color

To apply a text color, prefix your classes with text-.

<div class="text-teal text-darken-2"></div>
